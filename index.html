E-Commerce App

Open Project

Part of my coursework at the University of Houston was to work with a team to develop an e-commerce application. After explore various tech stacks, we ultimately decided to go with Django, due to its powerful features and ease of use. This application serves as the backend framework to handle the essential functionalities of the platform. The project includes a secure user authentication system, allowing customers to register,log in and place orders. It supports a comprehensive product catalog, featuring categories, product details, and inventory management. Key features include a dynamic shopping cart, where users can add, remove, and update products, and a seamless checkout process. The admin panel is customized to allow easy product management, order tracking, and user administration. To enhance user experience, the application implements responsive design principles, ensuring accessibility across various devices. Additionally, we utilized Django's templating engine to create clean and intuitive UI components, providing a smooth and engaging shopping experience.

Roulette App

Open Project

My friends and I were going to Las Vegas, but did not know the rules of any casino games. Roulette in particular, we found to be very confusing with the different types of bets and payouts. This inspired me to create a very basic roulette betting board application using only HTML, CSS, and JavaScript. This project was a fun exercise in building a functional web application without relying on any frameworks. The goal was to replicate a traditional roulette betting board, allowing users to place bets and see the potential payouts. After placing their bets and hitting the calculate button, the application displays the payout for each square. I focused on creating a clean and intuitive interface using pure HTML and CSS, while JavaScript handled the logic for calculating and displaying the payouts. This project helped me sharpen my skills in vanilla JavaScript and reinforced the importance of foundational web development techniques. It was a rewarding challenge to build a fully functional application from scratch, emphasizing simplicity and performance.
